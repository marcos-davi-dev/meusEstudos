Como instalar o BoothStrap ?

Existem duas formas de se utilizar a biblioteca do BootStrep, que são através da instalação da biblioteca usando o comando $npm install bootstrap@5.3.2 que é a versão mais atual, do dia de hoje 01/02/2024 .

Outra forma de se utilizar é via CDN que é um forma de utilziar todo arsenal do bootstrep sem precisar implantálo em seu servidor deixando ele mais leve.Para se utilizar basta colar dois links, um no < head > e outro no < body > . Na tag < head > utilizamos um link:

< link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous">

E na tag < Body > bem no final colocamos o script :

< script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous">

Depois de se aplicar os links em todas as tags podemos finalmente utilizar o BootStrap através da class, que são utilizadas para estilizar de forma mais pratica o nosso site.
