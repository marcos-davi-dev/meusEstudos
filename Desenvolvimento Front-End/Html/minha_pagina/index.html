Estrutura básica do HTML

A estrutura básica do HTML consiste em tecnicamente de 4 tags principais.
Temos a declaração < !DOCTYPE html > que é responsável por comunicar
ao navegador que essa é uma página HTML5. É importante estar sempre no
início do documento, antes da tag < html >.

As 3 tags que vem adiante são: < html > que define própriamente que estamos
iniciando um documento html; Temos o < head > que é onde iniciamos o cabe-
çalho do nosso site; E temos o < body > que é onde fica toda a parte visível do
nosso site.
